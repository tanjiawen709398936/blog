### 算法的效率问题：

为求解相同的问题而设计的不同算法在效率方面有显著的差别。

拿排序举例，排序 `n` 项
* 插入排序：所花时间大致等于 `cn^2`
* 并归排序：所花时间大致等于 `cnlgn`

其中 `c` 为不依赖 `n` 的常数当 n = 1000 时， lgn 大致为 10，当 n 等于 100万 时，lgn 大致为 20。对于小规模的排序：插入排序通常比并归排序要快，超过一定点，并归排序更快。


### 分治策略

1. 分解：分解成多个子问题
2. 解决：分别解决每个子问题
3. 合并: 将解决的结果合并

举例：一段时间（n天）内做一次股票交易怎样获益最大？也就是获取一个数组中值相加最大的子数组

解法1：暴力解决
1. 只要卖出日期在买入日期之后即可，n天共有 `n^2` 种组合

解法2：分治策略
1. 获取数组的中点 `n/2` （n/2需要取整）
2. 分解：一共 3 种情况
   1. 子数组在 0 至 n/2 之间（在左边） -> (n^2)/4
   2. 子数组在 n/2 至 n 之间（在右边） -> (n^2)/4
   3. 子数组在 跨越了中点 n/2（就是在中点两边） -> 2n
3. 一共需要 (n^2)/2 + 2n 次计算
